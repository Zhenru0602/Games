To run the jar flie:
Just double click GalaticMail.jar

To run the source code:
open galacticmail_source_code as a new project;
set the working directory to galacticmail_source_code;
run the main file tankgame_source_code/examples/KeyboardAnimation.java

Playing Instruction:
Moving the spaceship to different moon to score
Control: Up, down, left, right keys to move, space key to launch.
