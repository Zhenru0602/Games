To run the jar flie:
Just double click TankGame.jar;

To run the source code:
open tankgame_source_code as a new project;
set the working directory to tankgame_source_code;
run the main file tankgame_source_code/examples/KeyboardAnimation.java;

Playing Instruction:
Two player shooting game
Control: player 1: A, W, S ,D keys to move, Space key to shoot.
         player 2: Up, Down, Left, Right keys to move, Enter key to shoot.
